# Book-Market-Analysis
Proiectul vizează extragerea de informații valoroase și predicția succesului comercial în industria editorială, utilizând un set de date de pe platforma Kaggle. Obiectivul principal a fost aplicarea unor algoritmi de Machine Learning în limbajul R pentru a analiza impactul prețului, ratingului și notorietății autorului asupra veniturilor generate.
